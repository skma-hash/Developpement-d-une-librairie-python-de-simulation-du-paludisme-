# Developpement-des-librairies-python-de-simulation-du-paludisme-
Librairie Python dédiée à la simulation de la dynamique de transmission du paludisme, basée sur un modèle compartimental SVI–SI. Ce projet implémente un modèle à 17 compartiments et est un package open-source destiné à la recherche, à l’enseignement et à l’aide à la décision en santé publique.
